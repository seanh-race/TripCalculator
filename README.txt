This is a windows app.  Simply open TripCalculator.sln and start the program.  (Or build it and run the output)

Fill in the on screen prompts, and click the "Calculate Split" button to see the output in the text area below.

The Unit tests are in the UnitTestProject1.  Right click the project and click "Run Tests" to see them run.
